# Daily Proverbs App MVP

A working React/Vite prototype that creates daily Proverbs Scripture cards.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL Vite prints.

## What it does

- Daily Proverbs verse rotation
- Verse picker
- Theme picker
- Export wallpaper PNG
- Favorites using localStorage
- Share verse text
- KJV verse text included

## Important

This MVP uses KJV. NKJV generally requires licensing before using it in a public or commercial product.
